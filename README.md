# WeFIx

## Web Application

WeFIx is a web application for presenting a coffee products for a coffee provider, it shows the products in pretty way
with functionality of adding removing and updating it.

---

## Tools Used

VS Code

- TypeScript
- JavaScript
- HTML5/CSS3
- ReactJs

## Getting Started

Clone this repository to your local machine

```
$ git clone https://github.com/Jadaboawwad/thesis.git
```

Once downloaded, install the application dependencies by :

```
yarn install
```

Once installed, you can run the application by :

```
yarn start
```
